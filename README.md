# CS 4891 — Claude Code Course

An intensive, hands-on course on AI-assisted software development using Claude Code. Learn prompt engineering, agentic workflows, and modern development practices that leverage the power of large language models.

## 🎓 Course Overview

**Instructor:** Prof. Sophia Chen  
**Term:** Spring 2026  
**Credits:** 3 units  
**Format:** Hybrid (in-person + online)  
**Prerequisites:** CS 2110 or equivalent

### What You'll Learn

- **Prompt Engineering Fundamentals** — Craft effective prompts for code generation, debugging, and refactoring
- **Agentic Workflows** — Design multi-step AI agents that autonomously complete complex development tasks
- **Context Management** — Work efficiently with large codebases using optimal context strategies
- **Testing & CI/CD Integration** — Integrate AI assistance into automated testing and deployment pipelines
- **Production Best Practices** — Security, ethics, and responsible deployment of AI-assisted systems

## 📅 Course Structure

**14 weeks** covering:
- Weeks 1-2: Introduction & basic prompting
- Weeks 3-4: Prompt engineering & context management
- Weeks 5-6: Agentic workflows & multi-file editing
- Week 7: Midterm presentations
- Weeks 8-9: Advanced patterns, MCP tools, custom commands
- Weeks 10-11: Testing, debugging, CI/CD integration
- Weeks 12-13: Team workflows & code review
- Week 14: Final project presentations

**Grading Breakdown:**
- 40% Assignments (4 × 10%)
- 40% Final Project
- 10% Midterm Presentation
- 10% Participation

### Local Development

To view the site locally:

```bash
# Clone the repository
git clone https://github.com/gaojie058/claude-code-course.git
cd claude-code-course

# Open index.html in your browser
open index.html  # macOS
xdg-open index.html  # Linux
start index.html  # Windows
```

Or use a simple HTTP server:

```bash
# Python 3
python3 -m http.server 8000

# Node.js
npx serve
```

Then navigate to `http://localhost:8000`
