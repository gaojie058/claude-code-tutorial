Academic foundations

Reading Materials

Research papers and articles on AI-assisted development, language models, and software engineering.

Foundational Papers

"Attention Is All You Need"
Vaswani et al., 2017		 The Transformer architecture that powers modern LLMs. Essential background for understanding how Claude works. arXiv →
"Language Models are Few-Shot Learners"
Brown et al., 2020 (GPT-3)		 Introduces few-shot learning in large language models. Foundation for modern prompt engineering. arXiv →
"Chain-of-Thought Prompting"
Wei et al., 2022		 How to elicit reasoning in LLMs through step-by-step prompting. Critical for complex coding tasks. arXiv →
"ReAct: Reasoning and Acting"
Yao et al., 2023		 Synergizing reasoning with action-taking in language models. Foundation for agentic workflows. arXiv →

AI for Software Engineering

"Evaluating LLMs for Code"
Chen et al., 2021 (Codex)		 Introduction to evaluating language models on programming tasks. HumanEval benchmark. arXiv →
"AI Pair Programmers"
Ziegler et al., 2022		 How developers use AI coding assistants in practice. Productivity and adoption patterns. arXiv →
"Copilot Usage Analysis"
GitHub Research, 2022		 Large-scale study of GitHub Copilot usage. Developer productivity and code quality metrics. Blog →
"Security Risks in AI Code"
Pearce et al., 2022		 Analysis of security vulnerabilities in AI-generated code. Important for responsible deployment. arXiv →
Full citations and additional papers are available in the Canvas course module. Many papers are linked from weekly syllabus entries.
